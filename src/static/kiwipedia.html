Kiwipedia aka documentation

Files can be uploaded with the command line tool as well.

Simple upload:

$ curl --upload slack.dmg https://push.kiwi/slack.dmg
https://push.kiwi/HrfUfSAzj/slack.dmg

Email a file:

$ curl -H "x-email: jeffrey@lebowski.org" --upload slack.dmg https://push.kiwi/slack.dmg
https://push.kiwi/wYK1hNjCr/slack.dmg
