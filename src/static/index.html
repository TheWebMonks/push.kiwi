push.kiwi

Share files from the terminal.

# Share a file
$ push slack.dmg
https://push.kiwi/HrfUfSAzj/slack.dmg

# Email a file
$ push --email jeffrey@lebowski.org slack.dmg
https://push.kiwi/wYK1hNjCr/slack.dmg

# Share and protect with a PGP password
$ push --passphrase dude slack.dmg
https://push.kiwi/ubbTc74ul/slack.dmg.gpg

$
